# Puzzle_platformer
Платформер с головоломкой
В нашей игре нужно будет не просто пробежать уровень, а решить задачу. Например, нужно перенести ящик на кнопку, чтобы открыть дверь; использовать разные способности (двойной прыжок, лед/огонь) для взаимодействия с окружением. Состояние игрока (способности), управление сложными взаимодействиями между объектами, проектирование уровней с загадками
